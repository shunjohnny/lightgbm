相関係数の計算方法

相関係数マジ大事！-1から1の間の値をとるんだよ！

$$r = \frac{\sum{(x - \bar{x})(y - \bar{y})}}{\sqrt{\sum{(x - \bar{x})^2} \sum{(y - \bar{y})^2}}}$$
# Pythonでの計算方法 correlation = df['MEDV'].corr(df['RM'])

相関係数の見方！

  • 1に近い：超正の相関！（一緒に増える）
  • -1に近い：超負の相関！（片方増えると片方減る）
  • 0に近い：ほぼ無関係！

📊 評価指標を詳しく解説！

MSE (Mean Squared Error)

$$MSE = \frac{\sum{(y_i - \hat{y}_i)^2}}{n}$$
# 具体例で考えてみよう！ # 実際の値: 100, 予測値: 110 → 誤差: 10 # MSE = 10² = 100

二乗するから外れ値にめっちゃ厳しいの！

RMSE (Root Mean Squared Error)

$$RMSE = \sqrt{\frac{\sum{(y_i - \hat{y}_i)^2}}{n}}$$
# さっきの例だと # RMSE = √100 = 10 # 元の単位に戻るから分かりやすい！

MAE (Mean Absolute Error)

$$MAE = \frac{\sum{|y_i - \hat{y}_i|}}{n}$$
# 例で見ると誤差10の場合 # MAE = |10| = 10 # 外れ値があっても比較的影響小さい！

📈 正規分布について

標準偏差の意味

データのバラつき具合を表すよ！

$$\sigma = \sqrt{\frac{\sum{(x_i - \mu)^2}}{n}}$$
# μ（平均） = 22.53 # σ（標準偏差） = 9.19 # 約68%が μ ± σ (22.53 ± 9.19) 内に分布 # 約95%が μ ± 2σ (22.53 ± 18.38) 内に分布

🎯 判定係数（R²）の詳細

計算方法

$$R^2 = 1 - \frac{\sum{(y_i - \hat{y}_i)^2}}{\sum{(y_i - \bar{y})^2}}$$
# R²=1.0 → 完全予測 # R²=0.0 → 当てずっぽうレベル # R²<0 → 平均値予測以下...

注意点！

R²だけだと罠にハマることも！他の指標とも組み合わせて評価しようね！
